## Installation

### Clone the repository 

```sh
    git clone https://github.com/hroussille/agr4bs/tree/development
```

### Install the dependencies

```sh
    cd agr4bs
    git checkout development
    pip install -r requirements.txt
```

## Test suite

```sh
    python -m pytests 
```
